# Opslane Python SDK

The Opslane Python SDK will provide backend error capture for the Opslane error-resolution engine.

> **Pre-release scaffold — every API raises `NotImplementedError`.** Follow [opslane-oss issues #86–#89](https://github.com/opslane/opslane-oss/issues/86) for implementation progress.

```bash
pip install opslane
```
